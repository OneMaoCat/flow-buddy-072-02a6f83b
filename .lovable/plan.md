

# Code Review 布局改为垂直堆叠

## 现状
Done 状态下三个模型报告卡片是 **横向三列** (`flex gap-3`)，每张卡片宽度被压缩，findings 文字拥挤可读性差。

## 方案
改为 **纵向堆叠**，每个模型独占一个完整宽度的区块，从上到下依次展示报告。

### 具体改动（仅 `src/components/CodeReviewTab.tsx`）

1. **Done 状态布局**：`flex gap-3` → `space-y-3 flex-col`，每个 `DoneModelCard` 占满宽度
2. **DoneModelCard 样式调整**：
   - 移除 `flex-1 min-w-0`，改为 `w-full`
   - Header 区域：模型图标 + 名称 + 总结 + 分数环横向排列（宽度充裕后更舒展）
   - Findings 默认展示全部（空间够），超过 4 条再折叠
3. **Running 状态**：保持横向三列并排（体现并行感），不改动
4. **Overview 概览区**：保持不变
