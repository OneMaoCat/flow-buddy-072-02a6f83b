

# 问题清单卡片化：单问题轮播 + 隐藏其他元素

## 方案

将 `AcceptanceQA` 从一次展示所有问题改为 **单问题卡片轮播**，类似 `ClarifyCards` 的交互模式。同时，当问题面板显示时，隐藏 PromptBar 下方的任务列表按钮等多余元素，保持界面干净。

### 改动

| 文件 | 改动 |
|------|------|
| `src/components/DevCompleteDetailPanel.tsx` | 重构 `AcceptanceQA`：增加 `currentIdx` 状态，一次只渲染一个问题卡片；顶部加进度条指示器（参考 ClarifyCards 的圆点样式）；底部导航栏：上一个/下一个按钮 + 最后一题时显示确认按钮；用 `translateX` 实现滑动切换动效 |
| `src/pages/ProjectWorkspace.tsx` | 当 AcceptanceQA 显示时，隐藏底部 sticky 区域的任务列表 Popover 按钮和 PromptBar，只保留问题卡片 |

### 视觉效果

```text
┌──────────────────────────────┐
│  聊天消息流 ...               │
├──────────────────────────────┤
│  ● ● ○ ○          2/4       │  ← 进度指示
│                              │
│  ⚠ 警告 · LoginForm.tsx:42  │
│  密码字段缺少安全验证          │
│  建议：添加密码强度校验...     │
│                              │
│  ○ AI 修复(推荐) ○ 跳过 ○ 手动│  ← 选项
│                              │
│  [← 上一个]    [下一个 →]    │  ← 导航
└──────────────────────────────┘
```

最后一题选完后，"下一个"变为"确认并提交"按钮。

### 要点
- 复用 ClarifyCards 的进度条 + 滑动切换模式
- 问题面板可见时，`PromptBar` 和任务列表按钮隐藏，底部区域只有问题卡片
- 未选择选项时"下一个"按钮禁用
